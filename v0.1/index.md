This repository contains WDL files defining tasks. It is used as a central
repository for such task definitions within BioWDL and is used as a submodule
for most BioWDL pipelines and workflows. This way the tasks can be reused
between different workflows.

## About
This repository is part of [BioWDL](https://biowdl.github.io/)
developed by [the SASC team](http://sasc.lumc.nl/).
